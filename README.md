---

## Как запустить

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```
3. Откройте в браузере: `http://localhost:5173`.

---

## Возможности

- Отображение карточек курсов
- Поиск по названию (с debounce)
- Фильтрация по категориям (можно выбрать несколько)
- Кнопка сброса фильтров с анимацией перезагрузки
- Вращающаяся иконка при сбросе
- Полный адаптив до 320px
- Плавные hover-эффекты и анимации появления
